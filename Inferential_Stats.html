P value and statistical significance

What is a p-value? A great myth in social science is p-value. It is not an intuitive concept to wrap your head around. We use p-value to indicate statistical significance. The magic number here is .05: a p-value below .05 indicates a statistically significant relationship between variables. It indicates there is a 95% chance that the relationship found in the study sample is NOT due to random chances, and thus the finding from the sample accurately reflects the patterns in the population. Note that a p-value says nothing about the “strength of association,” or the meaningfulness of difference.

T-test

When to use t-test? T-test is a simple inferential statistical test used for comparing means across groups. It can reveal whether mean differences across different groups are statistically significant. You typically use t-test when you only have one categorical independent variable (e.g., sex) and one dependent variable.

Below we compare whether there is significant gender difference on the psychological construct called self-esteem.

Let’s inspect the data first. The SEX variable indicates whether a user is a male or female (1=male, 2=female). The column SELF_ESTEEM points to a measured self-esteem score reported in 1978 (higher value indicates a higher level of self-esteem).

# in this code block, we download the dataset from a web link and show the data table.

#show the first 50 observations

Run the t-test.

t.test(SELF_ESTEEM ~ SEX,data=data) #run the t-test

##  0.1814630 0.5819598
## sample estimates:

What is the p-value of the above t-test? Is it statistically significant? If see a p value higher than .05, it means that the test is NOT significant, thus the difference could be due to random change and unlikely reflect the true difference in the population. The 95 percent confidence interval gives you an estimate of a range of possible mean difference. Notice that 0 is inside of the range.

Correlation

When to use correlation? You can use correlation to test the strength of the relationship between two continuous variables. Note that the two variables must be continuous.

#test the correlation between self-esteem and wage reported in 1978

#ANOVA

How do sex and race predict self-esteem? When you have only one categorical independent variable, a test-test will suffice. With more than one categorical independent variable, we would need to use ANOVA. ANOVA is a statistical test used to predict a continuous variable from more than one independent categorical variable.

If you want to explore how SEX and RACE together are associated with SELF ESTEEM, note that we have two categorical independent variables here, we need to use ANOVA.

Before running a ANOVA model, it will be wise to explore the dependent variable to check its distribution. Is it normally distributed?

#explore the distribution of the DV
hist(data$SELF_ESTEEM)

Run a ANOVA model.

#Perform ANOVA

## make sure SEX and RACE are converted as categorical variables
data$SEX <- as.factor(data$SEX) 
data$RACE <- as.factor(data$RACE)

aov.1 <- aov(SELF_ESTEEM ~ SEX+RACE, data = data)
summary(aov.1)

Regression

How do SEX, RACE, and INCOME predict SELF-ESTEEM? Answering the question above requires using more than one independent variable and some of the independent variables are categorical and others are continuous.This is where regression comes into play. You use regression when your dependent variable is continuous and your independent variables are continuous or when your independent variables are mixed (i.e., some are continuous and others are categorical).

Perform regression
